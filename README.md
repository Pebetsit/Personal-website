# 🌐 Personal Portfolio Website

A responsive multi-page portfolio website built using **HTML** and **CSS**. The website showcases my background, technical skills, software development projects, research experience, publications, and contact information through a clean and professional design.

---

## 📖 Overview

This project was developed to strengthen my front-end web development skills while creating a professional online portfolio. It serves as a central platform to present my software development projects, academic background, research experience, publications, and technical skills as I transition from Physics into Software Engineering.
